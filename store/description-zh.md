# TempMail+ - 商店描述（中文）

## 扩展名称
TempMail+

## 简短描述（<=132 字符）
一键生成临时邮箱，保护真实邮件地址，防垃圾邮件，支持自动填写与深色模式。

## 完整描述

**TempMail+** 让你一键生成可用的临时邮箱地址，无需注册，即开即用。

### 核心功能
- **即时生成** - 1 秒内获得可用的临时邮箱
- **双服务商** - 主用 1secmail，遇到错误自动切换到 mail.tm
- **智能域名选择** - 优先使用曝光率低的域名，降低被屏蔽概率
- **域名信誉标识** - 直观显示域名是 🟢 冷门 / 🟡 中等 / 🔴 常见
- **自动填写** - 自动检测当前页面的邮箱输入框并一键填入
- **多邮箱管理** - 同时管理最多 5 个临时邮箱
- **自动轮询** - 每 10 秒自动刷新收件箱
- **深色模式** - 跟随系统或手动切换
- **中英双语** - 自动检测语言，也可手动切换

### 适用场景
- 注册网站时不暴露真实邮箱
- 开发者/测试人员批量账号注册
- 一次性查看邮件而不长期订阅

### 隐私说明
所有数据仅存储在浏览器本地（browser.storage.local），邮件内容不会上传至任何服务器。邮件直接通过邮件服务商的公开 API 获取。
